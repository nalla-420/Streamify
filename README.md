# Streamify 🎵
A free Spotify clone built with React.

## Deploy in 3 steps

### Option A: Netlify Drop (Easiest - no account needed)
1. Run: `npm install && npm run build`
2. Go to: https://app.netlify.com/drop
3. Drag the `build/` folder onto the page ✅

### Option B: GitHub + Vercel
1. Push this folder to a GitHub repo
2. Go to https://vercel.com → Import your repo → Deploy ✅

## Local development
```bash
npm install
npm start
```
